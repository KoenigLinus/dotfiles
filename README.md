# dotfiles

## Inhalt

- `.bash_profile`
- `.vimrc`
- `.zshenv`
- `.vim/`
- `.config/`
- `.sources/`
- `.scripts/`

## Initialisierung

```bash
cp -r $HOME/dotfiles/.vim ./
cp -r $HOME/dotfiles/.config ./
cp -r $HOME/dotfiles/.sources ./
cp -r $HOME/dotfiles/.scripts ./

ln $HOME/dotfiles/.bash_profile ./
ln $HOME/dotfiles/.vimrc ./
```
